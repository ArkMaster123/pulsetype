# Best Open-Source Mac Voice-to-Text Apps

Quick reference: speak → type, system-wide. All work in any app (Slack, email, editors, etc.).

---

## Top picks

### [FluidVoice](https://github.com/altic-dev/FluidVoice)

- **Download:** [Releases](https://github.com/altic-dev/FluidVoice/releases)
- **Site:** [altic.dev/fluid](https://altic.dev/fluid)
- **License:** GPL-3.0

Native macOS app. Write mode (dictate into any text box), command mode (voice actions), history, overlay. Parakeet / Apple Speech / Whisper. Optional AI polish (OpenAI, Groq). Offline-first, no account. Best if you want one app to use all the time and possibly extend.

---

### [open-wispr](https://open-wispr.com/)

- **Repo:** [human37/open-wispr](https://github.com/human37/open-wispr)
- **License:** MIT

Hold Globe (or custom key) → speak → release → text at cursor. 100% local (whisper.cpp + Metal), no accounts, no telemetry. Install via [install script](https://raw.githubusercontent.com/human37/open-wispr/main/scripts/install.sh) or Homebrew. Apple Silicon only. Best for minimal, “just works everywhere” setup.

---

### [Handy](https://handy.computer/)

- **Repo:** [cjpais/Handy](https://github.com/cjpais/Handy)
- **Download:** [Releases](https://github.com/cjpais/Handy/releases)

Push-to-talk or toggle mode, customizable shortcut. Fully offline, cross-platform (Mac, Linux, Windows). Simple and private.

---

### [OpenWhispr](https://openwhispr.com/)

- **Repo:** [OpenWhispr/openwhispr](https://github.com/OpenWhispr/openwhispr)
- **Download:** [openwhispr.com/download](https://openwhispr.com/download)

Open source, macOS / Windows / Linux. Cleans filler words. Free tier + Pro; can use your own API keys. Good if you want cross-platform and optional cloud.
